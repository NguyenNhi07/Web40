1. Với lập trình web: các khóa học đều đào tạo từ căn bản

Bạn là người thích và tư duy về cái đẹp, thích sự sáng tạo trong giao diện thì đây sẽ là khóa học dành cho bạn

Lộ trình Frontend + Reactjs- đây là khóa học để bạn có thể thiết kế được giao diện của 1 trang web

Bạn sẽ đc học vững các kiến thức cơ bản về HTML, CSS và javascript sau đó nâng cao là framework với ReactJs (đây là framework phổ biến hiện tại). Hoặc sau này bạn muốn chuyển qua lập trình di động đa nền tảng thì khi bạn hoàn toàn có thể bắt đầu với Flutter hay React Native.

Bạn là người thích thử thách hơn xíu: thích tư duy logic, thuật toán,… thì đây là lựa chọn của bạn:

Lộ trình Backend: Java Full Stack với Spring Boot 8 tháng

Bạn sẽ đc học cơ bản nhất là HTML, CSS sau đó sẽ học nâng cao lên Java Core, cấu trúc dữ liệu giải thuật, database, docker và đặc biệt là Spring Boot (đây là framework hiện hành bây giờ). Và ngôn ngữ Java cũng là ngôn ngữ Backend phổ biến hiện tại, có mức thu nhập cho ứng viên cũng cao hơn so với mặt bằng chung.

Khi bạn học Java sau này bạn muốn chuyển hướng qua di động, bạn có thể học Flutter (lập trình đa nền tảng), hoặc Kotlin (lập trình Android).
